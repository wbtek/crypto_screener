Â§Sorting Module

This module provides sorting functionality for the cryptocurrency data in the WBTek Crypto Screener application. The main function, sort_data, allows sorting of CryptoData entries based on a selected column in either ascending or descending order. The module includes specific comparison functions to handle different data types and ensure consistent ordering across various fields.
